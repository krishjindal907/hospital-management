# 🏥 Hospital Management System

A complete hospital management system built with React.js and Node.js.

## Features

- 👩‍💼 Reception Panel — Patient registration & token
- 👨‍⚕️ Doctor Dashboard — Live patient queue
- 💊 Prescription — Digital prescriptions with print
- 🔬 Lab Reports — Test results management
- 📅 Appointments — Online booking system
- 🧾 Billing — Bill generation with print
- 📊 Analytics — Charts and statistics
- 📱 Patient Portal — Self-service portal
- 🔐 Secure Login — JWT authentication

## Tech Stack

- Frontend: React.js + Vite
- Backend: Node.js + Express
- Database: SQLite
- Auth: JWT + bcrypt

## Setup

### Backend
cd hospital-backend
npm install
node server.js

### Frontend
cd hospital-app
npm install
npm run dev
